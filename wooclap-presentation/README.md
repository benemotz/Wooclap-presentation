
# RPSLS - Rock Paper Scissors Lizard Spock

## Author
**Motz, Bénédicte**

## Description
This is a presentation editing tool that enables users to:

- Navigate through slides
- Add notes per slide
- click and view predefined questions
- click and view the presentation tab

The app is built using React (via Create React App) and styled with custom CSS. It provides an intuitive interface for managing slides and interactive content.

## Features
- Slide Navigation: Browse through slides with previous/next controls and a thumbnail sidebar.
- Notes per Slide: Add and edit notes using a text area.
- Question Insertion: show questions with icons.
- Tabbed Sidebar: Switch between “Insert Questions” and “Updated Presentation”.
- Persistent Footer: Discard or save (doing nothing on clicking) with fixed footer buttons.

## Tech stack

- React 19 for UI components
- React Icons for question icons
- Custom CSS for styling
- ESLint + Prettier for code quality

## Setup Instructions

# Install dependencies
npm install

# Start development server
npm start

Open app in browser and go to http://localhost:3000
